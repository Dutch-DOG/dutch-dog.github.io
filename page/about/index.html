About

Dutch Dog

Welcome to the website for the Dutch DevOps & Github community!

Mission

We want to create a community for content related to DevOps and GitHub. We believe GitHub has a lot to offer and deserves a focus group in the Netherlands. But we also love the different elements that enable DevOps. So we decided to combine those two into the community you are looking at. We want to provide a platform where we can teach, learn and meet each other.

Schedule

We plan to have a meetup about once a month. This will start out fully virtual in light of the Corona virus, but will become partly IRL in the Netherlands in the future. The language will be English to be more inclusive.
